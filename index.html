How to use

Three steps and you're done

No coding, no fiddling with triggers — just pick a GIF and the mod handles the rest.

01

Place an anchor

In the level editor, select exactly one object. This is where the pixel art will be centered.

02

Pick your GIF

Click the import button in the Edit tab, hit Choose File, and browse to your GIF.

03

Play the level

The animation plays automatically. No trigger setup needed — it's all placed for you.

Features

What it actually does

Everything runs inside GD — no external tools needed once your GIF is ready.

🎞️

Full GIF decoding

Every frame is decoded and placed as individual pixel objects in the editor.

⚙️

Auto animation

Toggle and Spawn triggers are placed automatically to loop the animation in-game.

🧵

Background import

Large GIFs are processed in a background thread so the game doesn't freeze.

🎨

Accurate colours

Pixels are placed with HSV colour matching and transparency support.

📱

Android support

Works on mobile too, tested on Samsung devices. macOS and iOS builds included.

🔧

Tweakable settings

Control pixel size, animation speed, frame limit, group IDs and more from mod settings.

Settings

What you can change

All settings are in the Geode mod settings page. Defaults work fine for most GIFs.

Setting Default What it does
Frame limit 30 Max frames to import. Keep this low for big GIFs.
Animation speed 10 fps How fast the animation plays in-game.
Pixel size 5 Size of each pixel block placed in the editor.
Colour channel 1010 The colour channel used for pixel objects. Change this if it conflicts.
Base group ID 100 Starting group ID for frames and triggers. Raise this if it conflicts with your level.
Pixel limit per frame 10 000 Safety cap to prevent crashes on large GIFs. Can be disabled in settings.
