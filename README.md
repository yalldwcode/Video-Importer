# Video Importer

A Geometry Dash mod that lets you import video frames right into the level editor as pixel art. Built with the [Geode SDK](https://geode-sdk.org) and powered by [LuauAPI](https://github.com/ItMe12s/LuauAPI).

## How to use

1: Open the level editor and head to the Edit tab.

2: Select a single object to use as your anchor point (this is where the frames will start).

3: Click the import button, pick your video or GIF, and the mod will lay out the frames side by side.

## Settings
You can customize everything directly in the mod settings:

1: Frame limit and spacing between frames.

2: Pixel size and color channels.

3: Pixel limit per frame: This is a safety cap. Don't push this too high on huge videos or your game will probably crash!
